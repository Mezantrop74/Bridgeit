Speech

The 

speech
command allows you to have the device vocalize any provided text input and also accept any voice input from the user to be returned as text.

{% highlight javascript %} bridgeit.speech( 'myId', callback, {text: 'Why is the sky blue?', respond: true}); {% endhighlight %}
